# SmartTrack – Student Attendance Management System

## 📌 Project Overview

SmartTrack is a web-based student attendance management system designed to simplify the process of recording and tracking student attendance in educational institutions. The system allows teachers to add students, mark attendance, view attendance records, and generate attendance reports efficiently.

## 🎯 Objectives

* To automate the process of recording student attendance.
* To reduce manual errors in attendance tracking.
* To provide easy access to attendance reports for teachers.

## ⚙️ Features

* Add and manage student information
* Edit or delete student records
* Mark attendance (Present / Absent / Late)
* View attendance history
* Generate attendance reports
* Filter reports by subject and student

## 🛠 Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling and layout
* **JavaScript** – Functionality and logic
